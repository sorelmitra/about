The Friendly XML Editor (2000 – 2003)
XML was big at the time, and every large publishing house was moving its content to XML in order to be able to publish electronically.
Our company was busy editing and transforming XML content, and thus we built this editor in order to support their workflow.
And it did, effectively doubling their productivity. Because it could load very large files, even with syntax errors, and it out-performed every competitor out there.
Tech C++ Win32/MFC UML
AT Softwin SRL
The Mother of all Projects
This was an amazing project, in which we evolved a lot as developers, and created cool stuff.
Using background threads to build the display model in advance in order to enable loading of huge files.
Doing automatic testing in which we controlled the editor in all its aspects with a framework that we built.
Extreme programming, test driven development, refactoring, and pair programming.
And many more.
This project has set the foundation for me as a software developer. The techniques we applied here proved useful for every other project I was involved into.
Eyes (1997)
A fun project: implementing a pair of eyes that look at the mouse cursor, blink, and even go to sleep if you don't move your mouse for a while.
With configurable colors, intervals, and even focal distance. The latter affects how precisely the eyes follow the mouse, and makes them more fun.
A friend of mine helped me with some of the maths.
Tech Borland Delphi 2.0
A Word about Delphi
One of the best tools ever produced for Rapid Application Development, and one of my all-time favorite. I loved its instant compilation, excellent GUI editor, and the way it linked the UI editor to the code that you wrote behind it. Unparalleled ease of use and developer experience.
Mouse Emulator (1994)
One of my first projects ever, this was a general-purpose mouse emulator built for the famous, but mouseless ZX Spectrum.
You would use the keyboard to move a cursor around and ”click”. It would report the cursor position and button status into predefined memory locations.
Tech Z80 Assembly Language Zeus Assembler Editor
Hardware HC91 Computer B/W TV Set Antenna connection
Nostalgia
My first computer was a ZX Spectrum clone, named HC91 and built by the Romanian computer factory ICE Felix. I used to connect it to the antenna socket of a black and white TV set... I was in high school. I loved my HC91, and I have a certain nostalgia for that period in which you could live with 48K of RAM...
