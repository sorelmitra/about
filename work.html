Fractal Draw (1998)

Programmer - Borland Delphi 2.0

  • What does it do: It's an editor for visually defining and then drawing a fractal. The fractal is defined by first manually drawing a 2D "seed". Starting from the seed, the user can create multiple copies of it, which are initially identical to the seed. The user can rotate, size, skew the copy, and the program learns the relation between each copy and the seed. By activating the "draw" function, the fractal is drawn as follows: the seed is drawn first, then each copy is drawn according to its relation to the seed; then, each copy acts like the seed; and so on until the specified level is hit. The "paint" function applies dots directly at a very high level, considered "infinite"; an optional randomizing aids with realism. With properly defined fractals, the "paint" function produces realistic drawings.
  • My role: Conceive, implement, test.
  • What have I learned: Borland Delphi 2.0, debugging.
  • Context: Side project, during university.
  • Side notes: During that time, I've also used Delphi to create a pair of virtual eyes that follow the mouse cursor, blink, and fall asleep after an idle period; most of the parameters were editable.
